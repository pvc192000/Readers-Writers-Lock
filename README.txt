Instructions to run and modify the program:

To run the program:
  go to the folde containing the make file and
  $ make
  
  to run the project
  $ ./rwmain
  
  
To modify the project (edit the scenrios.txt)
Currently each line has 12 characters (each one either r or w)
To increase or decrease the number of characters per line:
  change the scenarios.txt file to represent new scenrios
  change the MAXLEN macro in the main.c file to the new number (currently 12)
